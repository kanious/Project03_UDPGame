1. How to build the project

- Please set "UDPServer" or "UDPClient" project as a starter project and build with x64 configuration / Debug or Release mode.
  Or you can execute with "UDPServer.exe" and "UDPClient.exe" file in x64\Debug(or Release) folder.

2. User Input Option

 < Player Sphere >
 * WASD : Moving Player Character(look/right direction)
 * QE : Rotate Player Character(by Y-Axis)
 * F : Fire bullet
 * R : Respawn

 < For Camera >
 You can look around the scene with this camera.
 * Arrow(↑↓→←) : Moving Camera (look/right direction).
 * Space/Left Shift : Up/Down Camera (up direction).
 * Alt : Enable/disable mouse. Rotating camera with mouse move when mouse is disabled.

 < Etc >
 * F1 : Reset Camera position to origin
 * ESC : Close the program.
